To create a one-page feel for my site, pagePiling.js allows the page to change site "pages" just by scrolling. In one html file, each "page" is defined as a div with the class "section". It also allows animation settings to be fine-tuned, so that horizontal scrolling can be simulated. Site sections can also be easily reached with its built-in navigation bar that tracks scroll position (like Bootstrap Scrollspy). This could help me create a site that users can navigate quickly and efficiently, all while having a cohesive experience.


I also searched for a plugin for the animated text on the landing page. I want the text to cycle between different words, in order to add life to the page and convey information in an engaging way. Both Morphext and Typed.js would work. They allow a String array to be specified and an animation to be chosen (e.g. fade or typewriter).
